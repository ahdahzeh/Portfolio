# Portfolio

Portfolio site (Next.js).

## Deploy on Vercel

1. Push this folder to GitHub (or GitLab/Bitbucket).
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import the repo.
3. Leave **Build Command** and **Output Directory** as default (Vercel detects Next.js).
4. (Optional) Set **NEXT_PUBLIC_SITE_URL** to your production URL if you use a custom domain and want correct social preview links.
5. Deploy.

Build: `npm run build`  
Start: `npm start`
