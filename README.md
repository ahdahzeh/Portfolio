# Portfolio

Portfolio site (Next.js).

## Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Push this folder to GitHub (or GitLab/Bitbucket).
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import the repo.
3. Leave **Build Command** (`npm run build`) and **Output Directory** as default (Vercel detects Next.js via `vercel.json`).
4. (Optional) In **Settings → Environment Variables**, set `NEXT_PUBLIC_SITE_URL` to your production URL for correct OG/social preview links.
5. Click **Deploy**.

Build: `npm run build`  
Start: `npm start`
